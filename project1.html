CSS Creature

Challenge Statement:
The challenge is to design a creature of your imagination. It could be anything you wanted as long as you could make it using html and css.

Solution:
For this project, I used majority css rather than html to make this image. For the html components of this project, I used a couple of body divs and messed around a lot with the css positioning. I was able to create this devil like creature by adjusting each elements in the devil to where I wanted it.
